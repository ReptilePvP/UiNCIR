
# CoreS3 Terp Meter Project (Codex Context)

## Project Overview
Temperature monitoring dashboard built using:

- M5Stack CoreS3
- PlatformIO
- LVGL 9.5.0 UI
- Pa.HUB v2.1 on Port A
- MLX90614 infrared temperature sensor
- M5Stack Joystick2 for UI navigation

The UI is written with LVGL and the joystick is used to navigate tabs and settings.
